📚 ClassMate AI

ClassMate AI is an AI-powered student assistant designed to help students solve academic doubts instantly.
The platform allows users to ask questions and receive intelligent responses using modern AI models.

It is built with React, TypeScript, and Vite and integrates AI APIs for real-time responses.

🚀 Features

✨ AI Doubt Solver
Students can ask academic questions and receive AI-generated explanations.

⚡ Fast Frontend
Built with Vite + React + TypeScript for high performance and modern development.

📦 Installation

Clone the repository:

git clone https://github.com/Himeshkar11/ClassMate.git
cd ClassMate

Install dependencies:

npm install
🔑 Environment Setup

Create a .env.local file in the root directory:

GEMINI_API_KEY=your_api_key_here

Replace with your Google Gemini API key.

▶️ Run the Project

Start the development server:

npm run dev

Then open:

http://localhost:5173

📚 ClassMate AI

ClassMate AI is a smart AI-powered study assistant designed to help students solve academic doubts instantly through an interactive chat interface.

The application provides a clean web interface where students can ask questions and receive AI-generated explanations in real time.

This project is built with React, TypeScript, and Vite, focusing on speed, simplicity, and a modern UI.

🚀 Features
🤖 AI Doubt Solver

Students can ask academic questions and receive explanations from an AI assistant.

💬 Chat-Based Interface

A conversational UI allows easy interaction with the AI model.

⚡ Fast Frontend

Built with Vite + React + TypeScript for extremely fast development and performance.

🧠 API Integration

Uses a backend API service layer (services/api.ts) to communicate with AI models.
